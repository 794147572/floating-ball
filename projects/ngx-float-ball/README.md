# ngx-float-ball

这是一个Angular的悬浮球控件。

- 支持触摸滑动
- 支持鼠标拖动
- 支持点击信号
- 支持设置鼠标点击延迟

## 安装

```
npm install ngx-float-ball --save
```

## 使用

1. 在app.module.ts内导入：

```
import { NgxFloatBallModule } from 'ngx-float-ball'
```

2. 在html内使用

```
<ngx-float-ball></ngx-float-ball>
```
